Hi, my name is Renato Bernardino and I made this slot machine for fun as a personal project.

Even though I'm not a gambler, since I am aware that I will probably just lose money, I really enjoy the design and mathematical domains of slot machines.

This slot is based in a game I like called Hunt Showdown. All the images and musics are from the game, if you want to check it out you can go to the game page here.

To know more information about the slot, press the button located in the bottom left corner of the slot, where you can learn about how much a line pays for each symbol, how the bonus feature works and also listen to the game musics.
